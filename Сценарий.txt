Сценарий:
Пользователь вводит три числа (Результат: вывод на экран корней/корня уравния(успешно), вывод ошибки)
Сценарий тестов:
Тест 1. Если все значения пользователя равны 0
	Результат: выводится ошибка 'R' в консоль, возвращается 	пустой массив
Тест 2. Если коэффиценты a и b равны 0
	Результат: вывод ошибки 'Нет корней' в консоль, 	возвращается пустой массив
Тест 3. Если a равно 0
	Результат: вывод одного корня 	уравнения
Тест 4. Если дискриминант уравнения меньше 0
	Результат: вывод ошибки 'Нет корней' в консоль, возвращаем 	пустой массив
Тест 5. Если дискриминат больше 0
	Результат: вывод 2-х корней в консоль
Тест 6. Если дикриминант равен 0
	Результат: вывод одного корня в консоль
